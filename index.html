JSON and XML Data Merge Tool

Project Overview Built a Python tool to merge and clean structured data across JSON and XML files. The workflow matches related airport records by code, updates outdated website fields, adds booking URL data from XML into matching JSON objects, saves a cleaned output file, and reports matched and unmatched records for review.

What It Does

  • Loads airport data from a JSON file
  • Loads update data from an XML file
  • Matches records using airport codes
  • Updates outdated website values
  • Adds new booking URL fields to matching records
  • Saves a cleaned JSON output file
  • Reports matched and unmatched records for review

Skills Used

Python JSON parsing XML parsing File handling Data transformation Record matching Output generation

Sample Outcome

The tool successfully updated matching airport records, added new booking URL fields where available, and flagged unmatched JSON and XML entries so they could be reviewed separately.

2 matching airport records updated Booking URLs added to matched records Unmatched JSON and XML entries flagged
