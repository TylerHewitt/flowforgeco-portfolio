How the cleanup work gets done behind the scenes

These portfolio examples are backed by actual workflows, not placeholder copy. The site now reflects the scripts and sample files behind the portfolio, including CSV cleanup, workbook generation, Google Sheets updates, summary reporting, and JSON normalization.

Flexible header matching

csv_normalizer.py and expense_cleaner.py look for alternate column names so files can still be processed even when headers vary.

Record-level cleaning functions

Names, emails, cities, dates, categories, descriptions, prices, and amounts are cleaned through focused helper functions before export.

Workbook generation

production_tracker_generator.py builds an Excel workbook with a titled sheet, bold headers, sample rows, and column sizing for readability.

Google Sheets updates

google_sheets_example.py uses gspread to update a sheet, write headers, and append rows for a collaborative tracking example.

JSON normalization

json_inventory_cleaner.py trims whitespace, title-cases item names and categories, and converts numeric text into cleaner structured data.

Summary reporting and QA

Text-based summary files capture counts, duplicates removed, missing values, spending totals, and category breakdowns so the cleanup work is easy to verify.

Workflow

How FlowForgeCo approaches cleanup work

01

Profile the raw file or sheet

Reviewed tabs, headers, field formats, blanks, and repeated values to understand where cleanup was needed.

02

Standardize structure

Normalized column names and aligned text, date, and number formatting so the dataset followed a cleaner pattern.

03

Remove duplicates

Checked for repeated entries and reduced the list to a more reliable set of records across spreadsheet rows or CSV records.

04

Summarize the outcome

Captured record counts, duplicate removals, missing values, and cleanup notes to support the final deliverable.
